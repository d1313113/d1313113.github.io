# 谷歌云配置 v2ray

由于近期又得知谷歌云可以申请,跟着教程申请了一个,并配置v2ray...

可以使用1年

# 准备工作

双币卡或者Visa信用卡,我使用的是招行master卡 Google账号 可以访问Google的梯子

# 申请 Google Could

  1. 进入Google Cloud页面

    谷歌云申请

根据参考资料中申请步骤申请好云

# 设置防火墙

  • VPC网络 -> 防火墙规则 -> 创建防火墙

    配置防火墙

# 设置静态IP

设置静态IP,防止每次重启实例后ip变动

  • VPC网络 -> 外部IP -> 保留静态地址

    配置静态IP

    需要注意几点:

    • 区域: 一般设置亚太区域,需要选择好地区,不然速度会很慢

    • 在设置好静态IP后记得先ping一下延时,延时在50ms以内比较合适,如果延时太大,可以重新新建,直到速度能接受,我的延时是15ms.

# 创建 VM 实例

  • Computer Engine -> 虚拟机实例 -> 创建实例

    创建实例

    需要注意几点:

    • 机器: 实例的费用不要超过$25/月,$300算下来每个月刚好$25

    • 地区: 选择刚上面配置静态IP的区域,对应好

    • 网络: 管理、磁盘、网络、SSH 密钥 -> 网络 -> 外部IP

      选择之前设置好的静态IP

# ssh 到 VM 实例

  • Computer Engine -> 虚拟机实例 -> SSH

    链接到VM实例,配置ssh登录

    在出来的命令行中,以此执行如下指令

    # 切换到 root 用户
sudo -i
# 更改 root 用户的密码
passwd
# 允许 ssh 登录
sed -i 's/PermitRootLogin no/PermitRootLogin yes/g' /etc/ssh/sshd_config
sed -i 's/PasswordAuthentication no/PasswordAuthentication yes/g' /etc/ssh/sshd_config
# 重启 ssh 服务
service ssh restart

# 安装 v2ray

  1. ssh到vm中
ssh root@ip
1
  1. 切换到root账号
sudo -i
1
  1. 安装v2ray
sh <(curl -L -s https://install.direct/go.sh)
1

执行成功后，会输出V2Ray v4.23.1 is installed.等字样。

  1. 修改v2ray配置文件 v2ray的配置文件在/etc/v2ray/config.json中
sudo vim /etc/v2ray/config.json
